- 1.3 `9.1` 已新增“接口归一化说明”，明确：
  - `social_cultural_signals` ← 1.1 PEST 社会文化环境归一化提取

  （1.3:571-572）
- 1.2 正式/条件输出包含：
  - `竞争机会清单`（1.2:677-690）
  - `定位图数据`（1.2:682-683）
  - `用户口碑洞察 C1`（1.2:692-699）
- 1.3 `9.1` 已明确：
  - `sentiment_available` ← 1.2 C1 是否可用
  - `top_unmet_needs / competitive_whitespace / opportunity_ids` ← 1.2 竞争机会清单、市场空白点、定位图空白区域归一化提取

  （1.3:573-574）

结论：
- 1.3 输入字段与 1.1/1.2 并非全部“原生同名直连”，但已被文档显式定义为“归一化提取/别名映射”，因此接口语义一致、无悬空字段。
