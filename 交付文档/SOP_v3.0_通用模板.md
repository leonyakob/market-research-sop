## 前置声明

### "求真务实"原则（5条规则）

AI在执行本SOP时，**必须严格遵循以下规则**：

1. **所有事实性信息必须来源于可验证的资料**（如学术文献、官方文档、权威出版物、可靠数据库、政府统计、上市公司财报）
2. **每一条关键结论应附带来源说明或推理依据**
3. **禁止生成无依据的推测、臆断或虚构内容**
4. **当缺乏足够信息支持时，应明确说明"信息不足"，而非补全或猜测**
5. **如存在多种观点，应说明不同来源及其差异，而非合并或简化为单一结论**

### P2级 - 建议完成（优化项）

- [ ] [优化项1]
- [ ] [优化项2]

1. **P0级全部通过**：可以提交报告
2. **P1级完成<80%**：添加"重要提示"章节说明未完成项
3. **P0级未通过**：停止执行，列出未通过项，等待用户决策
