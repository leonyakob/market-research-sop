## 前置声明

### "求真务实"原则（5条规则）

AI在执行本SOP时，**必须严格遵循以下规则**：

1. **所有事实性信息必须来源于可验证的资料**（如学术文献、官方文档、权威出版物、可靠数据库、政府统计、上市公司财报）
2. **每一条关键结论应附带来源说明或推理依据**
3. **禁止生成无依据的推测、臆断或虚构内容**
4. **当缺乏足够信息支持时，应明确说明"信息不足"，而非补全或猜测**
5. **如存在多种观点，应说明不同来源及其差异，而非合并或简化为单一结论**

### 3.3 AI自主补充查询

**补充触发条件表**：

| 条件 | 判断方法 | 阈值 | 示例 |
|------|----------|------|------|
| 数据缺口 | 关键指标缺失 | ≥2个核心指标无数据 | 市场规模、增长率缺失 |
| 数据冲突 | 多源数据差异 | 差异>20% | 来源A:100亿 vs 来源B:130亿 |
| 时效性不足 | 数据时间检查 | 最新数据>1年 | 最新数据为2023年 |
| 维度缺失 | 框架覆盖度检查 | 缺失≥1个必填维度 | PEST缺少T维度 |

**搜索失败处理流程**：

```
搜索失败 → 标记【信息不足】 → 记录影响评估 → 提供替代方案 → 继续执行
```

## 五、分析规则与决策标准

### 5.1 数据融合规则

**引用**：`SOP_v3.0_融合与冲突处理规则.md`

**融合流程**：

```
并行获取多源数据 → 智能比对 → 冲突检测 → 融合/冲突处理 → 输出结果
```

**冲突处理优先级**：

| 冲突类型 | 处理策略 | 优先级 |
|----------|----------|--------|
| 高可信 vs 中可信 | 采用高可信 | P0 |
| 多源高可信冲突 | 标注差异，说明采用依据 | P0 |
| 时效性冲突 | 采用最新数据 | P1 |
| 统计口径冲突 | 标注口径差异 | P1 |

### 5.2 竞争强度评估标准

**五力强度量化指标参考**：

| 维度 | 高 | 中 | 低 |
|------|----|----|-----|
| 现有竞争 | CR5<40%，价格战频繁 | CR5 40-60% | CR5>60% |
| 进入壁垒 | 资本需求低、品牌弱 | 资本中等 | 高资本、品牌强 |
| 替代威胁 | 高替代、转换成本低 | 中等替代 | 低替代 |

**关键成功因素优先级排序方法**：

```
评分维度：重要性(40%) + 稀缺性(30%) + 可获取性(30%)
排序规则：加权分高→低
```

### 5.3 生命周期判断标准

| 阶段 | 定性描述 | 量化参考指标 |
|------|----------|--------------|
| **导入期** | 新品类、增长快速但不稳定 | 增长率>50%，渗透率<5% |
| **成长期** | 快速增长、竞争开始加剧 | 增长率15-50%，渗透率5-20% |
| **成熟期** | 增长放缓、市场集中度提高 | 增长率5-15%，渗透率>20% |
| **衰退期** | 负增长或零增长、消费转移 | 增长率<0%，渗透率下降 |
